Privacy Policy

Last updated: March 29, 2026

Overview

ClipAI ("the Extension") is a browser extension that saves web pages with AI-generated summaries and tags to a local clip library. We are committed to protecting your privacy.

Data Collection

The Extension collects and processes the following data:

  • Website Content: When you click "Save This Page", the text content, title, URL, and favicon of the current page are read. The text is sent to OpenAI's API to generate a summary and tags. The summary, tags, title, URL, and favicon are stored locally in your browser.
  • Web History: URLs and titles of pages you explicitly save are stored locally in your browser as part of your clip library. We do not track browsing activity — only pages you choose to save.
  • Usage Counts: Monthly clip counts are tracked locally in your browser to enforce free tier limits (20 clips/month).
  • Subscription Data: If you subscribe to Pro, payment processing is handled by Stripe via ExtensionPay. We do not store your payment information.

Data Storage

All clip data (summaries, tags, notes, URLs), settings, and optional API keys are stored locally in your browser using Chrome's storage API. No personal data or clip data is stored on external servers. Our backend server only proxies AI requests and does not retain any content.

Third-Party Services

  • OpenAI API: Page text is sent to OpenAI for summary and tag generation. See OpenAI's Privacy Policy.
  • Cloudflare Workers: Our backend proxy runs on Cloudflare Workers. It forwards requests to OpenAI and does not store any user data.
  • ExtensionPay / Stripe: Payment processing for Pro subscriptions. See Stripe's Privacy Policy.

Data Sharing

We do not sell, trade, or transfer your personal information or clip data to third parties. Data is only shared with OpenAI for AI processing and Stripe for payment processing.

Data Export and Deletion

Pro users can export all clip data as JSON or Markdown at any time. You can delete individual clips from the Library tab. Uninstalling the extension removes all locally stored data.

Your Rights

You can uninstall the extension at any time to stop all data collection and delete all stored data. You can also clear stored data via Chrome's extension settings (chrome://extensions).

Contact

For questions about this privacy policy, please contact us through the Chrome Web Store listing.
